 ## Descripcion 

_Proyecto para el estudio de kanjis y diversas funciones para mejorar la memoria de los mismos con ayudade busqueda y deversas infromaciond e los mismos._

## tecnologias usadas 

* Reatc.js
* MongoDB 
* Express
* Redux (manjado desde redux toolkit)
* Css puro
